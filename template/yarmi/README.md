# React Native Authantication Template
> Clean and minimalist React Native template for a quick start an app with authantication with TypeScript.

## Features
- Multilanguage
- Save some information with Asyncstorage
- Redux
- Awesome components
- In App Template
- Absolute Path

## Installation
#### Clone repository

```sh
git clone https://github.com/ilkerkesici/react-native-starter-kit.git
```
#### Enter the folder

```sh
cd react-native-starter-kit
```
#### Run create_app.sh

```sh
./create_app.sh
```
#### Enter your app name

```sh
# Enter your app name :
MyAwesomeApp
```

#### Enter TS 

```sh
# Enter js for Javascript templates, ts for Typescript templates :  :
ts
```

#### Enter your template

```sh
# Enter template name :
yarmi
```
